* Beispiel:
  - Polarwinkel: $\theta = \frac{\pi}{2}$
  - Azimutwinkel: $\varphi$
  - Polarwinkel: $\theta = 2\beta$, $\beta = \frac{\theta}{2} = \frac{\pi}{4}$
```math
\begin{pmatrix}
\cos{\beta}\\
\sin{\beta} e^{i\varphi}
\end{pmatrix}
=
\begin{pmatrix}
\cos{\frac{\pi}{4}}\\
\sin{\frac{\pi}{4}} e^{i\varphi}
\end{pmatrix}
=
\begin{pmatrix}
\frac{1}{\sqrt{2}}}\\
\frac{1}{\sqrt{2}}} e^{i\varphi}
\end{pmatrix}
```
