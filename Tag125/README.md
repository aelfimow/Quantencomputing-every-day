Ab Seite 150:
* Mehrdeutigkeiten auf dem Äquator
  - alle Punkte auf dem Äquator: ein Qubit
  - für $\beta = \frac{\pi}{2}$:
```math
\begin{pmatrix}
\cos{\frac{\pi}{2}}\\
\sin{\frac{\pi}{2}} e^{i\varphi}
\end{pmatrix}
=
\begin{pmatrix}
0\\
e^{i\varphi}
\end{pmatrix}
```

* Phasenverschiebung: $e^{-i\varphi}$:
```math
e^{-i\varphi}
\cdot
\begin{pmatrix}
0\\
e^{i\varphi}
\end{pmatrix}
=
\begin{pmatrix}
0\\
e^{i(\varphi - \varphi)}
\end{pmatrix}
=
\begin{pmatrix}
0\\
e^{0}
\end{pmatrix}
=
\begin{pmatrix}
0\\
1
\end{pmatrix}
```
