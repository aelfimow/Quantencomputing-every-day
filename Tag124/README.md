Ab Seite 149:
```math
e^{-i\phi_{1}}
\cdot
\begin{pmatrix}
r_{1} e^{i\phi_{1}}\\
r_{2} e^{i\phi_{2}}
\end{pmatrix}
=
\begin{pmatrix}
r_{1} e^{i(\phi_{1} - \phi_{1})}\\
r_{2} e^{i(\phi_{2} - \phi_{1})}
\end{pmatrix}
=
\begin{pmatrix}
r_{1} e^{i\cdot 0}\\
r_{2} e^{i(\phi_{2} - \phi_{1})}
\end{pmatrix}
=
\begin{pmatrix}
r_{1}\\
r_{2} e^{i\varphi}
\end{pmatrix}
```

* Normierungsbedingung: $r_{1}^{2} + r_{2}^{2} = 1$
* $\cos{\beta} = \frac{r_{1}}{1} = r_{1}$
* $\sin{\beta} = \frac{r_{2}}{1} = r_{2}$
```math
\begin{pmatrix}
r_{1}\\
r_{2} e^{i\varphi}
\end{pmatrix}
=
\begin{pmatrix}
\cos{\beta}\\
\sin{\beta} e^{i\varphi}
\end{pmatrix}
```

Ab Seite 150:
* gilt für $\beta \in [ 0, \frac{\pi}{2}]$ und $\varphi \in [ 0, 2\pi ]$
  - d.h. obere Hemisphäre einer Kugeloberfläche mit Radius $1$
