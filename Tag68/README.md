Notizen zum Buch [Buch 1](../Buch1.md).

Ab Seite 49:
* Blochsphäre
  - $(\theta, \phi)$
```math
\begin{pmatrix}
\cos{\frac{\theta}{2}} \\
\sin{\frac{\theta}{2}}e^{i\phi}
\end{pmatrix}
```

* Nordpol: $\theta = 0$, $\phi = 0$ (oder beliebig)
```math
\begin{pmatrix}
\cos{0} \\
\sin{0}e^{i\cdot 0}
\end{pmatrix}
=
\begin{pmatrix}
1 \\
0
\end{pmatrix}
```
* Nordpol ist per Konvention:
$$
|0\rangle
$$

* Südpol: $\theta = \pi$, $\phi = 0$ (oder beliebig)
```math
\begin{pmatrix}
\cos{\frac{\pi}{2}} \\
\sin{\frac{\pi}{2}}e^{i\cdot 0}
\end{pmatrix}
=
\begin{pmatrix}
0 \\
1
\end{pmatrix}
```
* Südpol ist per Konvention:
$$
|1\rangle
$$
